## 功能
- 支持文案回显
- 支持一键扫描中文
- 支持文案上传到远程服务
- 支持拉取线上文案，定位文案的代码位置
- 支持翻译漏检功能
- 支持语言切换显示
- 支持分析统计
- 支持配置化，满足不同开发场景
